We have made available our security policy at [qiton.app/.well-known/security.txt](https://qiton.app/.well-known/security.txt).
This file contains information on how to report security vulnerabilities to us.
You can also [contact us](/miscellaneous/support) directly.

## PGP public key

We recommend to encrypt your message with our PGP public key.
This ensures that only the intended recipient can read the message.

PGP encryption may be tricky to use if you are not familiar with it.
A pretty good guide on how to use PGP is available [here](https://web.pa.msu.edu/reference/pgpdoc1.html).
