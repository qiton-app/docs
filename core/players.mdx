Rank and compare any player based on any stat available with the following filters:
- `Index`, the stat that determines how to rank players
- `Days`, a time filter
- `Rounds`, the minimum rounds played
- `Team`, the player's team
- `Agent`, the players that played the agent
- `Map`, the players that played in this map

## Prerequisites

- An active account ([learn more](/get-started/setup))

## Steps

Start by going to the players page at [qiton.app/players](https://qiton.app/players).
You will find all players detected by our system like below.
