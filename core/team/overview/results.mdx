This component provides another way to view the latest matches involving the selected team.
Clicking on any match will automatically redirect you to the corresponding match page.

## Prerequisites

- An active account ([learn more](/get-started/setup))
- You have to select a team beforehand ([learn more](/core/team/root))

## Steps

Navigate to the **Team** tab.
